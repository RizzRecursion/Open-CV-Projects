## 😄 Facial Expression Detection

A real-time facial emotion recognition app that classifies emotions like happy, sad, angry, etc. from your webcam.

## 🎥 Demo
![Facial Expression Demo](../assets/facial_expression.gif)

## 🧠 Model Info
- Trained on FER-2013 dataset
- CNN model (Keras/TensorFlow)
- Detects emotions: happy, sad, angry, surprise, neutral, etc.

## 🛠 Technologies
- Python
- OpenCV
- TensorFlow / Keras
- Pretrained CNN

## ▶️ How to Run
```bash
pip install opencv-python tensorflow keras numpy
python main.py
